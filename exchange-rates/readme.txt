== Installation ==

= From your WordPress dashboard =
1. Visit "Plugins > Add New"
2. Search for "Exchange Rates"
3. Activate "Exchange Rates" from your Plugins page.
4. Add widgets on yourdomain.com/wp-admin/widgets.php page.

= From WordPress.org =
1. Download "Exchange Rates".
2. Upload the "Exchange Rates" directory to your "/wp-content/plugins/" directory, using your favorite method (ftp, sftp, scp, etc...).
3. Activate "Exchange Rates" from your Plugins page.
4. Add widgets on yourdomain.com/wp-admin/widgets.php page.
